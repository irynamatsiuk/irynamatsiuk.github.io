Over Mij

Hallo, wat fijn dat u hier bent! Graag stel ik mij kort aan u voor. Mijn naam is Iryna, en sinds anderhalf jaar heb ik een sterke interesse ontwikkeld in webdevelopment.

Natuurlijk is dat niet mijn enige interesse. Ik houd ook van vogels, barefoot schoenen en wandelen. Eén van mijn mooiste persoonlijke prestaties is een 200 km lange kustwandeling langs de Costa Brava in Spanje. Ik vond het zó bijzonder dat ik deze route twee keer heb afgelegd.

Ik woon inmiddels bijna 10 jaar in Nederland. Na ongeveer vier jaar heb ik met succes het inburgeringsexamen afgerond – hoewel dit voor mij niet verplicht was. Ik heb het gedaan omdat ik het zelf belangrijk vond. De afgelopen drie jaar woon ik in Limburg, waar ik geniet van het heuvelachtige landschap.

Mijn Leerpad

Ik dacht altijd dat developers van een andere planeet kwamen — net als acrobaten bij Cirque du Soleil, die dingen doen die voor mij onmogelijk leken.

Dat veranderde toen ik een advertentie zag voor een online cursus 'Introduction to Web Development' van het Engelse initiatief Code First Girls (CFG). Op dat moment had ik hun cursus 'Introduction to Data & SQL' al succesvol afgerond, dus besloot ik deze ook een kans te geven.

De Web Development-cursus ging vooral over HTML en CSS. Dat was niet al te moeilijk, en gaf me het vertrouwen om verder te gaan. Vervolgens schreef ik me in voor 'Introduction to JavaScript', opnieuw via CFG. En daar eindigde het makkelijke gedeelte. Ik besefte al snel dat ik iets serieuzers nodig had dan een basiscursus die slechts aan de oppervlakte krabt.

Na wat onderzoek kwam ik tot de conclusie dat The Odin Project precies was wat ik zocht: zelfgestuurd, projectgericht en met een uitgebreide en goed opgebouwde leerlijn. Ik zal eerlijk zijn: het was (en is nog steeds) soms behoorlijk pittig en frustrerend. Maar hier ben ik dan — van nul kennis naar iemand die al een aantal toffe projecten heeft gebouwd!
Meer informatie over de cursussen die ik heb gevolgd vind u hier.

Mijn Projecten

De meest effectieve manier van leren is door te bouwen. Daarom zijn er in het curriculum van The Odin Project strategisch projecten opgenomen — en dat is meteen ook de belangrijkste bron van mijn portfolio.
U kunt mijn projecten van verschillende cursussen hier vinden, of bekijk een paar voorbeelden hieronder:

Homepage - een responsieve webpagina met drie breekpunten: voor mobiel, tablet en desktop. Gemaakt met pure CSS, zonder frameworks
Live Github

Shopping Cart - een single-page applicatie gebouwd met React Router. De gebruiker kan navigeren tussen Home, Shop en Cart, producten toevoegen of verwijderen uit het winkelwagentje. De styling is opgezet met CSS Modules.
Live Github

Members Only App: 'Friends' - een full-stack webapplicatie gebouwd met Express, PostgreSQL, EJS en SASS voor styling en responsiviteit. Voor meer details over elk project kunt u het README-bestand op GitHub bekijken.
Dit is mijn meest recente project. Houd er rekening mee dat ik nog maar halverwege het NodeJS-hoofdstuk ben — er valt dus nog veel te leren, zoals Prisma ORM, API’s, enzovoort.
De app is gedeployed via Render.com en heeft meestal 30 tot 60 seconden nodig om op te starten. Bedankt voor uw geduld! Live Github

De meeste projecten zijn gericht op het opdoen van ervaring in specifieke onderdelen. Als een project bijvoorbeeld draait om JavaScript, is de styling meestal eenvoudig gehouden. En andersom geldt: projecten met veel aandacht voor CSS bevatten vaak weinig JavaScript-interactie.
